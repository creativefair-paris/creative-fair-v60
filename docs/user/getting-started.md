# Premiers pas avec Creative Fair

## Bienvenue

Creative Fair est votre conseiller en communication digitale. Il vous
aide à transformer votre identité de marque en publications cohérentes,
chaque jour, sur Instagram.

Cette V1 est centrée sur quatre espaces principaux :

- **Aujourd'hui** — votre vue centrale, ce qui compte maintenant.
- **Calendrier** — vos publications, programmées et passées.
- **Ma marque** — votre identité, vos territoires, votre calendrier
  business.
- **Conseiller** — votre interlocuteur IA pour toute question
  stratégique.

## Première connexion

1. Ouvrez l'email d'invitation reçu après votre visio de cadrage.
2. Cliquez sur le lien magic-link pour activer votre compte.
3. Vous arrivez directement sur la page **Aujourd'hui** de votre
   tenant.

À votre première connexion, votre brand book est déjà pré-rempli sur
la base de votre visio. Vous pouvez l'affiner à tout moment dans **Ma
marque**.

## Que faire en premier

Trois actions à viser dans les 30 minutes qui suivent votre activation :

1. **Vérifier votre brand book** dans Ma marque → corriger ce qui
   doit l'être.
2. **Créer votre premier post** depuis le Calendrier (bouton
   « Nouveau post »).
3. **Programmer votre premier post** pour le lendemain matin.

## Ce que Creative Fair fait pour vous

- Génère vos contenus en respectant votre voix de marque.
- Croise votre actualité avec les bons rendez-vous calendaires.
- Vous propose des angles éditoriaux contextualisés.
- Vous prépare un brief externe pour un photographe ou rédacteur si
  besoin.
- Vous donne un avis stratégique via le Conseiller.

## Ce que Creative Fair ne fait pas (encore)

V1 :
- Pas de publication automatique sur Instagram (vous publiez
  manuellement, on vous prépare le post complet à coller).
- Pas d'analytics intégrés.
- Pas de support multi-canal (LinkedIn, TikTok, etc.) — Instagram
  uniquement.

Ces fonctionnalités sont prévues pour V2.

## Besoin d'aide

Contactez votre interlocuteur Creative Fair par email. Le support en
chat sera disponible V2.
