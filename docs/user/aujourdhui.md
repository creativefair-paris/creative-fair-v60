# Aujourd'hui

La page d'accueil de Creative Fair. Ce qui compte maintenant.

## Trois zones

### En-tête date

Affiche la date du jour, votre prénom, et un rappel discret du jour de
la semaine. Pas d'information critique ici — c'est un repère.

### Prochaine action

La carte centrale propose une action concrète, calibrée selon l'état
de votre tenant :

- Si votre brand book est incomplet → finir l'identité de marque.
- Si vous avez 0 post programmé pour les 7 prochains jours → créer
  votre prochain post.
- Si vous avez un post en brouillon (`status='ready'`) → le
  programmer.
- Sinon → un coaching IA personnalisé du jour.

### Coaching du jour

Un message personnalisé généré chaque matin par votre conseiller IA,
inspiré de votre brand book et de votre calendrier business. Si une
échéance importante approche (ex : Salon du Chocolat dans 7 jours), le
coaching la mentionne.

## Suggestions contextuelles

Une suggestion peut apparaître en haut de la page : par exemple
« Compléter votre brand book pour des contenus mieux ciblés ». Vous
pouvez la masquer (elle ne reviendra plus pour cette suggestion
précise).

## Crédits IA

L'indicateur de crédits du mois en cours est visible dans le bandeau
supérieur. Il décompose votre consommation par fonctionnalité
(coaching, génération, brief, conseiller).

## Pas suivants

Depuis la page Aujourd'hui, vous pouvez accéder en un clic à :
- Votre Calendrier
- Votre brand book
- Le Conseiller IA
