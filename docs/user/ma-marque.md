# Ma marque

L'identité de votre tenant. Le brand book et le calendrier business.

## Brand book

Votre carte d'identité éditoriale. Sept sections :

- **Identité** — nom, domaine, tagline, description, histoire.
- **Voix** — tons à privilégier, registre, mots interdits, mots
  encouragés, style des phrases.
- **Audience** — vos personas, leurs aspirations.
- **Territoires** — les grands thèmes éditoriaux (4-6 territoires
  recommandés).
- **Visuel** — palette, polices, références esthétiques.
- **Tabous** — tout ce qui ne doit jamais apparaître.
- **Objectifs** — ce que vous cherchez à atteindre globalement et sur
  Instagram.

Chaque champ est éditable. Les modifications sont prises en compte
immédiatement par l'IA pour vos prochaines générations.

## Calendrier business

Votre cadence métier. Quatre tableaux :

- **Événements récurrents** — ce qui revient chaque semaine, mois,
  trimestre, année.
- **Lancements à venir** — produits, événements, collections,
  partenariats programmés.
- **Saisonnalité** — niveau d'activité et thèmes clés par saison.
- **Événements industrie** — les rendez-vous de votre secteur (Salon
  du Chocolat, Fashion Week, Journée mondiale, etc.).

## Statut du brand book

Visible dans Aujourd'hui : **Complet** ou **Incomplet**. Un brand book
incomplet déclenche une suggestion contextuelle pour le finir.

## Coffres-forts

Vos données restent les vôtres. Aucune autre tenant ne peut les voir.
La séparation est garantie au niveau base de données par les Row Level
Security policies de Supabase.
